# lamp-php7.4.16
LAMP環境構築設定ファイル

## 説明
## 環境構成
- Docker 20.10.0以上
- Docker Compose 1.27.4
- PHP 7.4.16
- MariaDB 10.4.18 
- Apache 2系

## 利用方法
LAMPの環境構築に使用

## 実行
```
git clone https://github.com/fumiya5863/lamp-php7.4.16.git
cd lamp-php7.4.16
docker-compose up -d
```
